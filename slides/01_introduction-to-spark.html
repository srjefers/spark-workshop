Apache Spark

Distributed Big Data Computation Platform
Support for Scala, Java, Python, R and SQL languages
RDD - Distributed Parallel Scala Collections
Interactive Exploration
Batch and Iterative Workloads
Rich Data Source Support
Cluster Deployment with Standalone, YARN, and Mesos
Data Locality

Spark Core

  • RDD is the foundation of Apache Spark
  • Resilient
  • Distributed
  • Dataset
  • In-Memory, Immutable, Lazy Evaluated, Partitioned, Cacheable, Parallel, Typed

Spark SQL

  • Distributed SQL Engine
  • Main "programming" Language: SQL
  • Dataset (formerly: DataFrame)
  • Strongly-typed Query DSL
  • Hive Support
  • UDF - User-Defined Functions
  • Aggregation and Window Operators

Spark Streaming

  • Stream Processing
  • Discretized Stream (DStream)
  • Continuous Data Stream
  • Tight integration with other Spark modules

Spark MLlib

  • Large-Scale Distributed Machine Learning
  • Machine Learning Algorithms
  • Pipeline API
  • Classification, Regression, Recommendation, ...
  • Model Import / Export

Spark GraphX

  • Distributed Graph Processing
  • Property Graph (quite recently: GraphFrame)

Tools

  • spark-shell - an interactive shell for Apache Spark
  • spark-submit - a tool to manage Spark applications (i.e. submit, kill, status)
  • spark-sql - an interactive shell for Spark SQL and Hive queries
  • web UI - a web interface to monitor Spark computations
