What's wrong with the code?


     spark.range(10)
       .filter(_ % 2 == 0)
       .map(_ * 2)
       .filter(_ == 0)

What's wrong with the code? (2)


spark.range(10)
  .filter('id % 2 === 0)
  .map(n => n * 2)
  .filter('value === 0)
